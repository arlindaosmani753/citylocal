  <behavior>
    getFeedForCity(cityId, { category?, cursor?, limit? }):
    - Returns { posts: FeedPost[], nextCursor: FeedCursor | null }
    - Orders by createdAt DESC, id DESC (compound)
    - Excludes: deletedAt IS NOT NULL, status != 'active'
    - Excludes events where endsAt < NOW() (past events — EVNT-02)
    - category='event' → filters contentType='event' (not category column)
    - category='restaurant' (or any place category) → filters category column
    - category omitted/'all' → no category filter
    - Cursor: next page = rows where (createdAt < cursor.createdAt) OR (createdAt = cursor.createdAt AND id < cursor.id)
    - Fetches limit+1 rows; if len > limit → hasMore=true, slice to limit, return last row as nextCursor
    - Returns nextCursor=null when no more rows

    getPostsForMap(cityId):
    - Returns MapPin[] — { id, title, lat, lng } strings
    - Only contentType='place', status='active', deletedAt=null
    - Only rows where lat IS NOT NULL AND lng IS NOT NULL
    - .limit(200)

    getCityBySlug(slug):
    - Returns city row or null

    searchCitiesByName(query):
    - ilike(cities.name, '%query%'), limit 10
    - Case-insensitive match
    - Returns { id, name, slug, country }[]
  </behavior>
